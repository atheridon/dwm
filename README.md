# Atheridon's fork of dwm

The [suckless dwm](https://dwm.suckless.org/) with the following patches:

+ actualfullscreen
+ alpha
+ autostart
+ scratchpad
+ swallow
+ vanitygaps

## Configuration

Simply modify the `config.h` file. 
Default configuration is stored in `config.def.h`.

## Installation

```
sudo make clean install
```
